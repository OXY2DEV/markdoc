You can configure `markdoc` for a file by adding a YAML metadata section to
the start of the file.

It should look something like this,

>md
  ---
  markdoc:
    textwidth: 78
    title: "Markdown ğŸ¤ Vimdoc"
    title_tag: "markdoc"
  
    tags:
      Features$: [ "markdoc-features", "markdoc-feat" ]
      Usage$: [ "markdoc-usage" ]
  ---
  Some text.
<

â–Œ All configuration options should be inside `markdoc`.

>yaml

        border: "â•‘"

fold_refs ~

When `true`, link & image references are folded.

EXAMPLE

>yaml

<

foldmarkers ~

Markers used for folding.

EXAMPLE

>yaml

EXAMPLE

>yaml

        # The key is a Lua patter.

        Usage$: [ "markdoc-usage" ]
